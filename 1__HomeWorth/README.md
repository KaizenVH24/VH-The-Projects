# HomeWorth AI

Smart Property & Loan Intelligence System built using Machine Learning and Streamlit.

## Features
- House Price Prediction (10 Indian Cities)
- Future Price Forecast (3, 5, 7 years)
- Credit Worthiness & Loan Interest Estimator
- Clean Minimal Streamlit UI

## Tech Stack
- Python
- Scikit-Learn
- Pandas
- Streamlit
- Plotly
